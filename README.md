# 💡 Tcode 練習平台
這是一個類似 LeetCode 的線上練習平台，使用純前端 (HTML + JS + CSS)。

### ✅ 功能
- Python 語法編輯器 (Monaco)
- 自動補全（print, for）
- 題庫 (10 題含 Hard)
- 即時執行與模擬測資

### 🚀 部署
1. 上傳到 GitHub，repo 名稱需為 `tcod131.github.io`
2. 在 Settings → Pages 啟用  
   - Source：Deploy from branch  
   - Branch：main  
   - Folder：/(root)
3. 等約 2 分鐘後開啟  
   👉 https://tcod131.github.io/
